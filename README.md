# API

Wish & Complaint Backend

## Installation


Install the dependencies and devDependencies and start the server.

```sh
npm install
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

##  Error fixing the app 

```bash
# "NODE_ENV" is not recognized error
$ npm install -g win-node-env 
```
